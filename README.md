# Тема: Разработка онлайн-платформы "Танцевальные уроки" с использованием Scrum-методологии

В ходе выполнения дипломной работы была разработана и реализована
онлайн-платформа «Танцевальные уроки» с применением Scrum-
методологии, что позволило сформировать современный, гибкий и
ориентированный на пользователя подход к созданию цифрового
образовательного продукта. В процессе исследования проведён детальный
анализ предметной области: изучены особенности онлайн-формата обучения
танцам, определена целевая аудитория, выделены её потребности и
сформированы ключевые пользовательские требования.

На этапе проектирования была сформирована архитектура будущей
платформы, определена структура её функциональных модулей, построены
диаграммы вариантов использования для ролей «Пользователь» и
«Администратор», а также разработана контекстная диаграмма IDEF0,
отражающая основные процессы взаимодействия системы с внешними
элементами. Итогом проектирования стала разработка полноценного веб-
приложения, включающего главную страницу с фильтрацией уроков по
танцевальным стилям, личный кабинет с профилем пользователя и
привязанными банковскими картами, страницу оплаты с выбором
сохранённых карт или вводом новых, а также страницу просмотра урока с
видеоплеером, описанием и домашним заданием.
